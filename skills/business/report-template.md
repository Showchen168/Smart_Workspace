## 【領導匯報原則】

### 金字塔原則
```
        結論
       /    \
    論點1   論點2
   /  \     /  \
 證據  證據 證據 證據
```

1. **先說結論**：開門見山，不要鋪陳
2. **再說論點**：支持結論的 2-3 個主要論點
3. **最後是證據**：數據、案例、事實

### STAR 法則（用於說明成果）
- **S**ituation：背景情況
- **T**ask：任務目標
- **A**ction：採取行動
- **R**esult：達成結果

### 數字說話原則
| 避免 | 改為 |
|------|------|
| 大幅提升 | 提升 35% |
| 很多客戶 | 127 位客戶 |
| 節省成本 | 節省 NT$ 50 萬 |
| 反應很好 | 滿意度 4.8/5.0 |

## 【使用方式】

告訴我以下資訊，我會生成對應的匯報文件：

1. **報告類型**：週報/月報/簡報
2. **報告對象**：誰會看這份報告
3. **主要內容**：要匯報什麼
4. **時間範圍**：報告涵蓋的期間
5. **特殊需求**：有無特別要強調的內容
